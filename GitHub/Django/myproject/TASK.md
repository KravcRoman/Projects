# Создание приложения блога

## Требования к приложению:
1. Создание модели и миграции:
Создайте модель Django для простого блога. Модель должна содержать поля для 
заголовка, текста и даты публикации статьи. Напишите миграции для этой 
модели.

2. Создание и отображение шаблона:
Создайте шаблон Django для отображения списка статей блога. Выведите заголовки и даты публикации статей на этой странице.

3. Добавление формы:
Добавьте
 форму на страницу создания новой статьи блога. Форма должна содержать 
поля для заголовка, текста и даты публикации. Напишите представление, 
которое будет обрабатывать эту форму и создавать новую статью в базе 
данных.

4. Добавление аутентификации:
Используйте встроенные 
средства Django для добавления аутентификации пользователей. Ограничьте 
доступ к странице создания новой статьи только для аутентифицированных 
пользователей.

5. Добавление комментариев:
Расширьте модель 
статьи блога, добавив возможность комментирования. Создайте модель 
комментария, которая будет связана с моделью статьи. Напишите 
представление и шаблон для добавления комментариев к статье.

6. Добавление тестов:
Напишите
 тесты для ваших представлений, моделей и форм. Убедитесь, что тесты 
покрывают основные случаи использования и проверяют корректность работы 
вашего приложения.
