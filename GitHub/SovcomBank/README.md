Задание 1: Долевое строительство

Алгоритм:
Прочитаем количество долей и сами доли в виде дробей (например, 1/2, 3/4).
Для каждой доли умножим её на 100, чтобы перевести в проценты.
Выведем результат с точностью до трех знаков после запятой.

Почему так работает:
Мы просто умножаем дробь на 100, чтобы перевести её в проценты.
Для точности до 3 знаков используем формат вывода, который обрезает лишние знаки.

Сложность:
Время: Процесс занимает O(N), где N — количество долей.
Память: Используем O(N) памяти для хранения долей.
Для N до 10^6 алгоритм работает быстро.



Задание 2: Мегатрейдер

Алгоритм:
Прочитаем количество дней, максимальное количество лотов и доступную сумму.
Для каждого дня проверим, можем ли мы купить облигации на предложенную сумму.
Рассчитаем доход от купленных облигаций.
Выведем общий доход и информацию о купленных лотах.

Почему так работает:
Мы покупаем облигации, если у нас достаточно денег, чтобы купить целый лот.
Доход считается из номинала облигаций (1000 единиц) с учетом их стоимости и количества.

Сложность:
Время: Для каждого дня проверяем все лоты, что требует O(N*M), где N — количество дней, M — количество лотов.
Память: Для хранения всех лотов потребуется O(N*M) памяти.
Для N и M до 1000 решение работает быстро.
