Необходимо:

Получить данные по комментариям и постам с ресурса http://jsonplaceholder.typicode.com/

http://jsonplaceholder.typicode.com/posts
http://jsonplaceholder.typicode.com/comments
Посчитать среднее количество комментариев к посту каждого пользователя, результатом должен быть словарь формата:

user_id
average_comments_per_post
Результат вывести в stdout (например print).
