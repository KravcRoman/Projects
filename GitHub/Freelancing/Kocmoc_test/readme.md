Написать скрипт/приложение, который обрабатывает файлы exel (смотри приложенные файлы).
Результатом работы скрипта должен быть файлы типа . recs
Имя:  номер счетчика из строки 5 (начиная с колонки D) + год и месяц расчетного периода. Например 80512135_2023_13. recs
Файлов должно получится столько сколько номеров счетчиков указано в файле 
Содержимое файла  
- Первая строка: номер счетчика затем через запятую показания за первый день расчетного периода 
- Вторая  строка: номер счетчика затем через запятую показания за второй день расчетного периода 
- Количество строк = количеству дней в расчетном периоде

+ Если все необходимые данные существуют добавление записей в таблицу (создание таблицы  в файле txt)  
Num – номер счетчика
Base – пусто
Dat (n-й день расчетного периода)
Devtyp – пусто
Если показания введены за час то заполняются  v001, v011, v021 и т.д
Если показания введены за полу часы, то показания вводятся в каждую колонку

При обработке файла проверять, если все данные существую (номер и в месяце показания на все даты и часы, то при перемещении файла в папку  wrk добавлять _0
Если данных не достает, то помечать _ и код ошибки (составить список кодов)
